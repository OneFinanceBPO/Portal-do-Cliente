Política de Privacidade

Como coletamos, utilizamos e protegemos seus dados pessoais em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

Última atualização: junho de 2026  ·  Versão 2.0

1Controlador dos Dados

O controlador dos dados pessoais tratados neste portal é a One Finance BPO Financeiro, responsável pelas decisões relativas ao tratamento de dados pessoais de seus clientes e usuários.

One Finance BPO Financeiro
Contato do DPO (Encarregado de Dados): privacidade@onefinance.com.br

2Dados Pessoais Coletados

Coletamos apenas os dados estritamente necessários para o funcionamento do portal:

  • Dados de identificação: nome completo e endereço de e-mail para acesso ao portal;
  • Dados de autenticação: hash criptográfico da senha (nunca a senha em texto claro);
  • Dados de uso: registros de login, logout e ações realizadas no sistema (log de atividades);
  • Dados da empresa: razão social, CNPJ e segmento de negócio para fins de identificação do cliente.

Não coletamos dados sensíveis, dados de menores de 16 anos, nem dados de geolocalização.

3Finalidade do Tratamento e Base Legal

Os dados são tratados com as seguintes finalidades e bases legais (art. 7º da LGPD):

  • Execução de contrato — autenticação e controle de acesso ao portal financeiro contratado;
  • Legítimo interesse — segurança da informação, prevenção a acessos não autorizados e auditoria de ações;
  • Obrigação legal — manutenção de registros exigidos pela legislação fiscal e contábil aplicável;
  • Consentimento — envio de comunicações relacionadas a atualizações e melhorias do portal (opcional).

4Armazenamento e Segurança

Os dados são armazenados em repositório privado no GitHub (Microsoft Azure), com acesso restrito por chave de API e token de autenticação. Medidas de segurança implementadas:

  • Senhas armazenadas como hash SHA-256 — nunca em texto claro;
  • Sessões com expiração automática por inatividade (20 minutos);
  • Comunicação exclusivamente via HTTPS/TLS;
  • Acesso ao repositório restrito por tokens com escopo mínimo necessário;
  • Proxy Cloudflare Worker para proteção do token de escrita no GitHub.
Prazo de retenção: Os dados de perfil são mantidos enquanto o contrato de prestação de serviços estiver ativo. Logs de atividade são retidos por até 90 dias no dispositivo local (localStorage). Após o encerramento do contrato, os dados são excluídos em até 30 dias.

5Compartilhamento de Dados

Os dados não são vendidos, alugados ou cedidos a terceiros para fins comerciais. O compartilhamento ocorre apenas com:

  • GitHub, Inc. (Microsoft) — provedor de infraestrutura de armazenamento de dados, com sede nos EUA (transferência internacional amparada por cláusulas contratuais padrão);
  • Cloudflare, Inc. — provedor de proxy de borda (Worker), utilizado como intermediário seguro para operações de escrita;
  • Autoridades públicas — quando exigido por obrigação legal ou ordem judicial.

6Direitos do Titular dos Dados

Nos termos do art. 18 da LGPD, você possui os seguintes direitos:

  • Confirmação e acesso — confirmar se seus dados são tratados e obter cópia;
  • Correção — corrigir dados incompletos, inexatos ou desatualizados;
  • Anonimização, bloqueio ou eliminação — quando os dados forem desnecessários ou tratados em desconformidade;
  • Portabilidade — receber seus dados em formato estruturado e interoperável;
  • Revogação do consentimento — a qualquer momento, para os tratamentos baseados em consentimento;
  • Oposição — opor-se a tratamentos realizados com base em legítimo interesse;
  • Reclamação à ANPD — peticionar perante a Autoridade Nacional de Proteção de Dados.

Para exercer seus direitos, entre em contato pelo e-mail privacidade@onefinance.com.br. Responderemos em até 15 dias úteis.

7Cookies e Armazenamento Local

Este portal não utiliza cookies de rastreamento ou analíticos. O armazenamento no dispositivo é utilizado apenas para:

  • sessionStorage — dados de sessão do usuário autenticado (apagados ao fechar a aba);
  • localStorage — configurações de sincronização, log de atividades local e token de autenticação com o GitHub (quando configurado manualmente pelo administrador).

Nenhum dado pessoal é transmitido para serviços de análise, publicidade ou redes sociais.

8Alterações nesta Política

Esta política pode ser atualizada periodicamente. Alterações relevantes serão comunicadas por e-mail aos usuários cadastrados com antecedência mínima de 10 dias. A versão vigente estará sempre disponível nesta página com a data de atualização.
